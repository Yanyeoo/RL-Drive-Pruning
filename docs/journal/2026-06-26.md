### 14:00–15:40  Phase 2 v0 R1'' probe (Task C) + dataset prep (Task A)

**目标**：在 V4 sweep 启动前的 cheap 时间窗内验证 design doc §10 的两个核心假设：
- R1''：**跨层 attention（L0/4/8/16/20/24）能预测 L12 bot-K head 选择吗**？
- L24 mask 的 bot-K 频率是不是真的 dominate（V4 选 7/9/10 是否过强或过弱）？

**做法**：
- **Task A — 数据集**：用 `exp/m1b2_navtrain_full_alllayers/*.pt` 19,225 个 `(28,16,720)` attention tensor，对每个 token：(a) X = L{0,4,8,16,20,24} per-head attention（layer × head 平均到 vision tokens），(b) y = L12 bot-K head set (k=4)。落到 `exp/m1b2_phase2_v0/dataset.pt`，分 train/val/test (80/10/10)。耗时 ~13 min。
- **Task C — Probes**：P1 (linear 96→16) + P2 (MLP 96→64→16)，二者均在 1-2s CPU 内训完 + eval。
  - P1 EM=0.30，等同于"const top-4 baseline"
  - P2 EM=0.30，**完全打平 baseline，跨层信息为 0**
- **Task D — 单 K (K=4) bot-K freq scan**：跑了 7 个 layer 的 single-shot bot-K freq，**L24 bot-K 顶 3 是 h9 (100.0%), h10 (98.9%), h7 (93.7%)，与 V4 mask 完全一致**（V4 mask 是用 rank-variance 通过独立路径选的）。

**结论**：
- **R1'' 死** — design doc §10 Q1 "use L12 hidden as scene rep" 是对的选择 (R1, not R2/R3)
- **V4 mask 被独立 freq 数据强力 confirm**
