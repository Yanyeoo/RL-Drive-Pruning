## 20:00 — 20:30  user 对齐: framing 重写为 per-head binary probe (走法 1)

4. user 原话: **"我觉得只要效果好 都行你懂吗。走法1也行吧 主要看效果 你持续更新吧"** → mandate

**结论**: 抛弃 two-stage (R1 学 K → R1pp 选 head)，转 **single-stage per-head binary probe**：
- Input: token feature (96-d，复用 Phase 2 R1pp dataset)
- Output: 16 维 independent sigmoid → 0/1 mask
- Loss: `L_task + λ · sum(mask_keep)` (λ 越大越鼓励多删)
- λ-sweep → dynamic (K_eff, PDMS) Pareto 曲线，与 static 曲线对比

**新 Gate G_p3_1'** (Pareto dominance, 任一即可):
- (a) 同 K_eff 下 dynamic PDMS 更高 ("选对了 head")
- (b) 同 PDMS 下 dynamic K_eff 更大 ("选对了 token: 该多删的多删")
- (c) 双赢

**R1 降级**: 进 ablation 表当 baseline (two-stage vs single-stage)，**不再当 inference path**。
