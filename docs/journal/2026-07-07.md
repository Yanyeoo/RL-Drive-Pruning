### 根因分析（scorer r=0.5 vs r=1.0 全 navtest, −0.687pt）

**核心发现**：掉点完全由 **1.3% 灾难场景** 驱动，不是普遍退化。

| 场景分类 | N | 占比 | 说明 |
|---|---|---|---|
| scorer ≥ r=1.0 | 10,352 | 89.4% | scorer 无损或更好 |
| scorer < r=1.0, 非灾难 | 1,064 | 9.2% | 小幅退化 |
| **scorer=0 且 r1>0.8（真正灾难）** | **155** | **1.3%** | **决定总 delta 的关键** |

关键数字：
- **P50 delta = 0.0**（中位场景完全无差异）
- 排除 155 个灾难场景后：scorer mean = 0.938, r=1.0 mean = 0.931, **scorer 实际领先 +0.7pt**
- 155 灾难场景的 r=1.0 baseline 平均 0.9523（原本很好的场景）→ 被 scorer 杀到 0
- 这 155 场景贡献了 **−1.275pt**（>总 delta −0.687pt，因为剩余场景是正的）
- random 在这 155 场景得 0.63（也受损但不是 0 → scorer 做了极差选择）

**诊断**：scorer(4000 navtrain 训练) 在少数 OOD 场景上的 token ranking 极度错误，
选中了完全无关的 token、丢弃了关键决策 token，导致 trajectory 生成崩溃。

**可行修复方案（价值排序）**：
1. **(高价值/低成本) Safety-net fallback**：推理时检测 scorer confidence/entropy；当 scorer 不确定时回退 r=1.0（仅 1.3% 场景多用 token，平均 keep_ratio 仅从 0.5 升到 ~0.507）
2. **(中价值/中成本) 扩训 scorer**：用全 19k navtrain 重训（当前只用 4000），更好覆盖 OOD
3. **(论文 framing) 不修复，诚实报告**：claim 改为"iso-compute r=0.5 下 dominate attn/random；98.7% 场景无损，headroom 来自 safety-net / 扩训"

## 21:35 — Safety-Net 方案 FAIL + 更深层诊断

**Safety-net 基于 scorer uncertainty 不可行**（详见 `2026-07-07_decision_safety_net.md`）：
- scorer 在灾难场景上 entropy=0.36 vs 正常=0.36 — 无差异
- boundary gap 同样无差异（0.0134 vs 0.0135）
- **scorer 是 "confidently wrong"，不是 "uncertain"**

**更深层根因（按 baseline 难度分层，全量 N=11571）**：

| 场景类型 | N | 占比 | scorer vs r1 |
|---|---|---|---|
| r1=0（baseline 灾难）| 505 | 4.4% | **+18.7pt** ✅ |
| r1<0.8（困难）| 750 | 6.5% | **正增益** ✅ |
| r1∈[0.8,0.95)（较简单）| 5017 | 43.4% | **−1.2pt** |
| r1>0.95（满分区）| 5804 | 50.1% | **−2.0pt** ← 主退化 |

**总结**：claim① 的"贴边"不是少数灾难场景的问题那么简单，更本质的是——**50% 的 navtest 场景本身就是"满分"场景（r1>0.95），对这些场景任何 50% 剪枝都会引入 ~2pt 的微扰退化**。这是 50% token reduction 的结构性代价，safety-net 无法解决。

**最终策略（方案 D）**：
1. **论文 framing**：不说"lossless"，说"iso-compute dominate all baselines"+ "困难场景大幅受益"
2. **等 attn_L12 r=0.5 数据**：如果 attn_L12 掉得更多（很可能），scorer 仍 dominate → claim③ 完美成立
3. **scorer r=0.75 Pareto 点**：只剪 25% 时退化应极小，提供 Pareto 曲线让 reviewer 选
4. **扩训 scorer（下窗口）**：可减少灾难场景数量，改善 overall 约 0.3-0.5pt
